Functional Requirements

1. Create navigation menu (containing button to main dashboard, settings, info)

2. Add ability to view previously created counters

3. Add ability to create new counter with following options: counting voters / counting bands / counting either voters or bands

4. Add counter component with user actions: add, remove, leave coment or go to info

5. Add information component containing 2 tabs: graph with hour step / list of items with time

6. Add navigation to comments page

7. Add ability to edit comments

8. Add long tap on day tile to delete it

9. Add light and dark theme to app

10. Add theme swithcer to settings

11. Add localization to the application

12. Add translation (Belarusian, English, Russian)

13. Add languager selector to settings

14. Add vibration on add, remove voter/band

15. Add vibration selector to settings

16. Add info about developers to info page so that user could contact us in case of any questions

Non-Functional Requirements

System should meet the following non-functiaonal requirements:

  • - be secure
  • - be correct
  • - be quick
  • - be free
  • - be accessible
